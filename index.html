iOS Installation Guide

Simple steps to get started on iPhone/iPad

1

Download Profile

Safari browser မှတဆင့် Configuration Profile ကို ဒေါင်းလုတ်လုပ်ပါ။

2

Trust Certificate

Settings > General > VPN & Device Management တွင် Trust လုပ်ပေးပါ။

3

Install & Enjoy

Home Screen တွင် ပေါ်လာသော App ကိုဖွင့်ပြီး ကြည့်ရှုနိုင်ပါပြီ။

Android TV & Android Box များအတွက်

Android TV နှင့် Android Box များတွင် YLA TV ဆော့ဝဲလ် ထည့်သွင်းနည်း အသေးစိတ်လမ်းညွှန်

အဆင့် (၁)

Downloader App ကို Install ပြုလုပ်ပါ

Google Play Store သို့ဝင်ရောက်ပြီး Downloader ဟုရိုက်ရှာပါ။ လိမ္မော်ရောင် Icon လေးနှင့် App ကို Install ပြုလုပ်ပေးပါ။
